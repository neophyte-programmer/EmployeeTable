
# EmployeeTable

## Installing

```
- git clone https://github.com/quo-vadis/EmployeeTable.git
- cd InterviewTask.UI
- npm install
- cd ..
- open InterviewTask.sln with Visual Studio to run WebApi server. 
- right click your project in the Solution Explorer and choose Properties,
  then navigate to the Web tab and under Start Action choose: 
  Don't open a page. Wait for a request from an external application
- start project (F5)
- npm start
```
